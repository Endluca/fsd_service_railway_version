访问凭证
自建应用获取访问凭证
功能：自建应用通过此接口获取 app_access_token
提示

1. app_access_token 的最大有效期是 2 小时。
2. 如果上一个app_access_token的有效期大于 30 分钟，调用本接口会继续返回上一个app_access_token,并且不刷新有效期； 
3. 如果有效期小于 30 分钟，调用本接口，会返回一个新的 app_access_token ，这会同时存在两个有效的 app_access_token。

接口地址 Request URL：/openapi/auth/v1/app_access_token/internal
接口方法 Request Method：POST
限流等级说明：Light
请求头 Request Headers
参数	描述	是否必须	类型	备注
Content-Type	固定值："application/json"	是	string	 
入参说明
请求体 Request Payload

参数	描述	是否必须	类型	备注
app_key	应用唯一标识	是	string	创建应用后获得，参考获取应用凭证
app_secret	应用秘钥	是	string	创建应用后获得，参考获取应用凭证
请求体示例

{
    "app_key": "cli_slkdjalasdkjasd",
    "app_secret": "dskLLdkasdjlasdKK"
}
返回体数据data
名称	描述	类型
app_access_token	自建应用访问凭证	string
token_type	认证类型	string
expire	自建应用过期时间	int
响应体示例

{
    "code": 0,
    "msg": "success",
    "data": {
        "app_access_token": "a-6U1SbDiM6XIH2DcTCPyeub",
        "token_type":"Bearer",
        "expire": 7140
    }
}
业务错误码
http状态码	code	msg	描述	排查建议
400	400050	appSecret invalid	无效的应用密钥	查看app_key和app_secret是否有效
